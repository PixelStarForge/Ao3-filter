# AO3 Filter

**Block unwanted fanworks on Archive of Our Own with custom CSS.**

AO3 Filter is a browser extension that helps you hide works you don't want to see on AO3 — by author, by tag, or by individual fic. It generates custom CSS that you can paste directly into your AO3 Site Skin.

---

## Why This Exists

If you've spent any time in the Re:Zero fanfiction community (or any fandom, really), you've probably noticed the flood of:

- Anonymous NTR and Rape fics
- Ragebait and low-effort content
- AI-generated slop

This extension was built because i was tired of it and worst things was you could not even block anonymous fics or when you were reading as a guest.

---

##  Features

- **Author Blocker** – Hide all works from a specific author.
- **Tag Blocker** – Fetch tags from any work and select which ones to block.
- **Block Fics** – Load all works from an author and selectively hide individual fics.
- **Copy & Download** – Copy generated CSS to clipboard or download as a `.css` file.
- **Search** – Filter through fics with fuzzy search.

### Manual Installation (Developer Mode)

1. Download or clone this repository.
2. Open Chrome and go to `chrome://extensions/`.
3. Enable **Developer mode** (toggle in the top right).
4. Click **Load unpacked** and select the extension folder.
5. The extension icon will appear in your toolbar.

##  How to Use

Click the extension icon in your browser toolbar to open the popup.

### Author Blocker

1. Paste an author's profile URL (e.g., `https://archiveofourown.org/users/username`).
2. Click **Generate CSS** or press `Enter`.
3. Alternatively, click **Block Anonymous Fics** to hide all anonymous works.
4. Copy the generated CSS and add it to your AO3 Site Skin (see below).

### Tag Blocker

1. Paste a work URL (e.g., `https://archiveofourown.org/works/12345`).
2. Click **Fetch Tags** or press `Enter`.
3. Select the tags you want to block.
4. Copy the generated CSS and add it to your AO3 Site Skin.

### Block Fics

1. Paste an author's profile URL.
2. Click **Load All Fics** or press `Enter`.
3. All works from that author will be loaded, grouped by fandom.
4. Select the individual fics you want to block.
5. Copy the generated CSS and add it to your AO3 Site Skin.

## Applying the CSS to AO3

AO3 Filter generates CSS rules that you need to add to a **Site Skin** on AO3. Here's how:

### Step-by-Step Guide

1. **Log in** to your AO3 account (you must be logged in to use Site Skins).
2. Go to your **Dashboard** by clicking your username in the top right corner.
3. Click the **Skins** link.
4. Click the **Create Site Skin** button.
5. Give your skin a **unique title** (e.g., `My AO3 Filter`).
6. In the **CSS** box, paste the CSS code generated by the extension.
7. Click **Submit** at the bottom of the page.
8. Go back to your **Dashboard** → **Skins** and click **Use** on your new skin to activate it.


### Important Notes

- Site Skins only work when you are **logged in** to AO3.
- You can have multiple Site Skins and switch between them.
- If you want to use the CSS alongside an existing skin, you can paste it into that skin's CSS box instead of creating a new one.
